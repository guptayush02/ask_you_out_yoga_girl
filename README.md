# Setup

## If you have python installed:
```
cd ask_you_out_v
```

&& 

```
python -m SimpleHTTPServer --port  8081
```

visit http://localhost:8081 in your browser.

## If you have nodejs installed
```
npm install
```
&&

```
npm run server-node
```
visit http://localhost:8081 in your browser.
